📸 Instructions pour capturer l'image Open Graph

  1. Option 1 - Capture d'écran manuelle :
    • Zoomez votre navigateur à 100% (Ctrl+0 / Cmd+0)
    • Utilisez un outil de capture (Snipping Tool Windows / Screenshot Mac)
    • Capturez UNIQUEMENT le rectangle noir ci-dessus (1200x630px)
    • Sauvegardez sous le nom og-image.png
  2. Option 2 - Outil en ligne (recommandé) :
    • Ouvrez la console de votre navigateur (F12)
    • Cliquez droit sur l'élément avec id="og-image"
    • Sélectionnez "Capture node screenshot" (Chrome/Edge)
    • L'image sera téléchargée automatiquement
  3. Remplacez l'ancienne image dans : public/og-image.png
  4. Rebuild le site : npm run build
  5. Déployez la nouvelle version
  6. Forcez le rafraîchissement du cache Facebook sur : https://developers.facebook.com/tools/debug/
